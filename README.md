# generate-password

## Description
A website that helps generate a password based on criteria that the user chooses. 

## Criteria
* User chooses password length of at least 8 characters and no more than 128 characters
* Prompts user to choose to add uppercase letters, lowercase letters, symbols and numbers
  * At least one character type should be selected
* Once all prompts are answered then a password will generate in the box

## Built with
* HTML
* CSS
* JavaScript (only worked on JavaScript)
